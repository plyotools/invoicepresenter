# Change Log

This file tracks all changes made to the project. Each entry should be very short and descriptive.

## Format
- Date and time: `YYYY-MM-DD - HH:MM: Description`
- One line per change
- Focus on what was done

## Examples
- `2025-12-24 - 23:15: Removed Filter column`
- `2025-12-24 - 23:16: Added Change log feature`
