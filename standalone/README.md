# Standalone Invoice Presenter

This folder contains a self-contained HTML file that can be opened directly in any web browser.

## How to Use

1. **Open the file**: Simply double-click `index.html` or open it from your file manager
2. **Or drag and drop**: Drag `index.html` into any web browser window
3. **Upload your Excel file**: Click "Upload Excel file" and select your `.xlsx` or `.xls` file
4. **Use the app**: The table will display with all the invoice data

## Features

- **No server required**: Works completely offline
- **No installation needed**: Just open the HTML file
- **Self-contained**: All CSS and JavaScript are embedded in the single file
- **Works in any browser**: Chrome, Firefox, Safari, Edge, etc.

## File Size

The `index.html` file is approximately 810KB and contains everything needed to run the application.

## Building

To rebuild the standalone version, run:

```bash
npm run build:standalone
```

This will regenerate the `standalone/index.html` file with the latest changes.
